# Environments

![Extension](https://files.lezi.uno/ext.png)

# Commands
1. `copy .env.example .env`
2. `composer update`
3. `composer install --no-dev --optimize-autoloader`
4. `php artisan key:generate --force`
5. `php artisan migrate`
