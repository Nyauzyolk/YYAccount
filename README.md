# YYAccount
A Simple Third-Party Auth System By Nyauzyolkūüėä

# Run
## Environments
Apache or Nginx + PHP 8.2 + MySQL 5.7.44 + Composer 2.4.1
## Commands
1. `cp .env.example .env`
2. `composer install --no-dev --optimize-autoloader`
3. `php artisan key:generate --force`
4. `php artisan migrate`
5. `php artisan serve`
